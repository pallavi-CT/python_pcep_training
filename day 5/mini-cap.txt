Build a configurable risk scoring system for loan applications:

Write individual scorer functions: score_credit(credit_score), score_income(monthly_income, loan_amount), score_employment(years_employed), score_debt_ratio(total_debt, income)
Each scorer returns a value 0–100 (higher = lower risk)
Write calculate_risk(*scorer_fns, weights=None, **applicant_data) that:
- Calls each scorer with the applicant data
- Applies weights (default: equal weight)
- Returns composite score and per-component breakdown
Write make_decision(composite_score) → "Approve" / "Review" / "Decline"
